Jump into the ultimate interactive blocks companion! This is a beautifully crafted encyclopedia and interactive dashboard designed for every Minecraft explorer. From detailed mob guides and spawn charts to interactive mini-games like a clicker and silhouette quizzes, this app brings the world of Minecraft straight to your browser with premium animations and responsive styling.

## 🚀 Live Website

Experience the fully interactive website live here:
https://minecraft-iota-six.vercel.app

## 🎮 How to Control

* **Explore Mobs:** Dive into detailed stats, behaviors, and health points for various creatures.
* **Silhouette Quiz:** Test your Minecraft knowledge by guessing mobs solely from their shadow.
* **Mob Clicker:** Click on mobs, gather XP, and unlock achievements!
* **Spawn Table:** Browse spawn rates, behaviors, and drop lists in an interactive data table.
* **Crafting & Brewing:** Play around with simulated recipes.

## ⚙️ Installation

### Prerequisites
* Node.js (v18 or higher)
* npm

### Setup Instructions
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
